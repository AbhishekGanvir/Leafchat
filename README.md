# Run Locally

### Setup .env file

```js
DATABASE_URL=your_postgres_url..
JWT_SECRET=your_secret..
NODE_ENV=devlopment..
PORT=3000..
```



### Build the app

```shell
npm run build
```

### Start the app

```shell
npm run start
```
